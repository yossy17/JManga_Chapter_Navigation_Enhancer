このスクリプトは、[Jmanga](https://jmanga.org)で左右矢印キーを使用してページチャプターの前後を可能にします。

## 使い方

- 左右の矢印キーでページチャプターの前後ができます。：
  - `←`：前のチャプターへ移動
  - `→`：次のチャプターへ移動
